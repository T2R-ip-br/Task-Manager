# Система управления задачами

Веб приложение позволяющее:
  - Добавлять задачи
  - Добавлять подзадачи
  - Отображать все задачи в виде дерева
  - Контролировать трудоёмкость, время выполнения задач и их статус
  - Редактирвоать задачи
  - Удалять задачи
